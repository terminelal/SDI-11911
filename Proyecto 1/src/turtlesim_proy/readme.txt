Para ejecutar el programa es necesario que tanto ROS como turtlesim estén funcionando, para ello se deben seguir los siguientes pasos:
1. Abrir una terminal y ejecutar el comando roscore
2. Abrir una nueva terminal sin cerrar la anterior y ejecutar el comando rosrun turtlesim turtlesim_node

Una vez iniciados ROS y turtlesim y descargada la carpeta del repositorio: https://github.com/sebosanchez/robotica
1. Irnos a la carpeta descargada (cd CARPETA_DESCARGADA)
2. Ahora debemos enseñarle a la computadora donde está nuesto ejecutable, para eso corremos dentro de la carpeta el siguiente comando: source devel/setup.bash
3. Corremos el programa con el siguiente comando: rosrun turtlesim_proy proy_node  
5. Seguimos las instrucciones en pantalla para ejecutar el movimiento
